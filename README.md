# Bank_Loan_Analysis_Project-PowerBI

## Project Overview

This project presents a comprehensive Bank Loan Analysis dashboard built using Power BI. The goal is to analyze loan-related data and derive insights that help understand loan performance, customer behavior, and potential risk areas.

## Objectives

* Analyze loan applications, approvals, and disbursements
* Track key performance indicators such as total loans, approval rates, and default trends
* Identify risk patterns across customer segments

## Key Features

* Interactive dashboards with slicers and filters
* KPI cards for quick performance monitoring
* Trend analysis by loan status, purpose, and customer profile
* Segmentation based on income, employment, and credit-related attributes

## Tools & Technologies

* **Power BI** – Data modeling, DAX, and dashboard creation
* **Microsoft Excel / CSV** – Source data

## Insights Covered

* Approved vs rejected loan trends
* Loan status and repayment behavior
* Customer risk segmentation
* Loan distribution across categories

## Use Case

This project is suitable for data analysts, business analysts, and finance professionals seeking hands-on experience with Power BI and financial data analysis.
